# Smith & Associates Law Firm - Sample Data

This directory contains sample data for Smith & Associates, a fictional law firm based in the United States.

## Directory Structure

```
smith-associates/
├── auth/               # Authentication and user data
├── cases/              # Case management data
├── clients/            # Client information
├── documents/          # Document metadata
├── billing/            # Invoices and billing data
├── calendar/           # Court dates and appointments
├── tasks/              # Task management
├── time/               # Timesheet entries
├── expenses/           # Expense tracking
├── payments/           # Payment records
├── trust/              # Trust accounting
├── accrual/            # Accrual accounting
├── accounting/         # General ledger and accounting
├── referrals/          # Referral management
├── workflows/          # Workflow templates
├── email/              # Email templates and data
├── reports/            # Report templates
├── settings/           # Firm settings and configuration
├── security/           # Roles and permissions
├── portal/             # Client portal data
├── customer-portal/    # Customer portal specific data
├── projects/           # Project costing data
├── ai/                 # AI assistant data
└── audit/              # Audit logs
```
